Folder & HTML Structure

sree is one page coming soon HTML site template. So, mostly you will need one .html file & supporting .css, .js, .php files with some images. Structure of your the template would look like following image.

  • All .html files are in the root directory.
  • Separete folders are created for .css, .js files & for images.
  • All fonts other than web-fonts are located in fonts folder.
  • Mailgun folder is for sending transactional emails.
  • Important .php files for sending emails are located in root folder.
What do you need to upload to server?

You will need to upload .html file along with supporting folders provided with this theme.

General css classes:

  • .light-header: For light header
  • .dark-header: For dark header
  • .animsition: For animated page transition. Generally added to body tag. You can read more at http://git.blivesta.com/animsition/.
  • .header: For placing Logo & social media links
  • .container, .row, .col-*-*: Bootstrap classes for layout & grid. Read more on http://getbootstrap.com/
  • .mb*: For margin-bottom. Eg. mb20 for margin-bottom: 20px;
  • .mt*: For margin-top. Eg. mb20 for margin-top: 20px;
  • .pb*: For padding-bottom. Eg. mb20 for padding-bottom: 20px;
  • .pt*: For padding-top. Eg. mb20 for padding-top: 20px;
  • .trans: For all CSS transitions
  • .cd-section: For one section with full height on medium & large monitors
  • .sr-btn: For styled button
  • .sr-btn.sr-primary: For styled button with primary color

Color Schemes & Headers

We have included 8 different color schemes with this theme. You can simply change css file & apply new color scheme to your website.

CSS file name tells you which color it is. You have alizarin.css, carrot.css, midnightblue.css, nephritis.css, peterriver.css, sunflower.css, torquoise.css & wisteria.css. Just add this CSS file instead of main.css to apply new color scheme.

Mailchimp Setup

Warning!

Before uploading template to your hosting server, make sure you have done correct setup for Maichimp Subscription list & contact us form. It is easy and properly documented in this file.

Mailchimp setup is easy but mandatory step. If you already have Maichimp account & a list, just go to sign up forms for the particular list. Get form action URL from embedded forms section. Paste this in following section of script:

$('.mc-form').ajaxChimp({
    url: '', //Enter your URL here
    callback: callbackFunction
});

Steps:

  1. Signup / Login on Mailchimp.
  2. Create a list.
  3. For this list, go to 'Signup Forms' section.
  4. Go to 'Embedded Forms'.
  5. Copy action url from the form. [Shown in below gif image.]
  6. Paste it in your html file in scripts section.

Make sure you have added following two js files.

<script src="js/jquery.ajaxchimp.min.js"></script> <!-- For ajax Mailchimp submission -->
<script src="js/sweetalert.min.js"></script> <!-- For js alerts -->

Contact Us Form Setup

We have integrated mailgun for uninterrupted transactional emails. You can use this free service for 10,000 emails every month. Visit mailgun.com for more information.

It is already integrated and ready to use. You just need to put your domain details & api-key in settings file.

  1. Create account on mailgun.com for free.
  2. You can use sandbox domain that you get or you can add your own domain with some simple steps provided at their website.
  3. You will need two important factors from your mailgun account. 1. Domain Name. 2. API Key
  4. Click on 'domains' link from navigation.
  5. Select the sandbox domain.
  6. On this screen on mailgun.com, you will get both details.
  7. Open mailgun-settings.php file which is included in your theme's downloaded folder.
  8. Paste mailgun domain name & API key here.
  9. Enter your email address in $yourEmail variable, where you want to receive emails from website. Replace support@themesease.com with your email id.
  10. You can change additional settings for email in same file. These are optional changes.

Make sure you have uploaded 'Mailgun' folder provided with your theme download.

CountDown Setup

Adding countdown is easy. Make sure you have added following css in head: 

<link rel="stylesheet" href="css/jquery.countdown.css" media="screen">

Make sure you have added following js files at end of body.

<script src="js/jquery.plugin.min.js"></script>
<script src="js/jquery.countdown.min.js"></script>
Below these lines, add following code in script tag: [Refer original files from download for reference.] 
$('#countdown').countdown({until: new Date(2016, 12-1, 01), format: 'wdHMs'});

Important!

A note on Date - the JavaScript Date constructor expects the year, month, and day as parameters. However, the month ranges from 0 to 11. To make explicit what date is intended (does a month of 3 mean March or April?) I specify the month from 1 to 12 and manually subtract the 1. Thus the above date denotes 01 December, 2016.

Refer http://keith-wood.name/countdown.html for more information and settings.

Word Rotator Setup

We use morphext library for rotating words. jQuery and animate.css files are required for this one to work. Make sure you have included both of them with morphext.css and morphext.min.js.

Add this line where you want to show word rotator:

<h1 class="headline">We believe in <span id="word-rotate">Perfection, Create, Impress, Enthrall</span>. <small>We are coming soon.</small></h1>

Words mentioned within id="word-rotate" and separated by comma will rotate when you add following script:

// For Word Rotator
$("#word-rotate").Morphext({
    animation: "flipInX",
    separator: ",",
    speed: 4000
});

You can use different animations with these words easily. Just mention animation in above code in-place of flipInX. You can use bounce, flash, pulse, shake etc. Refer http://morphext.fyianlai.com/ for more details.

Static & Slideshow Images

You can easily change images in the background slideshow. We use backstretch library for background slideshow.

Don't forget to add <script src="js/jquery.backstretch.min.js"></script>. To change images, use below code:

/*
* Here is an example of how to use Backstretch as a slideshow.
* Just pass in an array of images, and optionally a duration and fade value.
*/

// Duration is the amount of time in between slides,
// and fade is value that determines how quickly the next image will fade in
$.backstretch([
  "http://dl.dropbox.com/u/515046/www/outside.jpg"
, "http://dl.dropbox.com/u/515046/www/garfield-interior.jpg"
, "http://dl.dropbox.com/u/515046/www/cheers.jpg"
], {duration: 3000, fade: 750});

Just put your image urls replacing above ones. This script is in script section at end of the HTML file.
